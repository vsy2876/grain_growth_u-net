# Grain Growth U-Net
Generative diffusion models for predicting the thermodynamic evolution of Potts models (grain growth).

## Models Included
- **Direct Jump:** t0 to t_final prediction.
- **Next Timestep:** Autoregressive step-by-step prediction.
- **LSTM:** Temporal memory integration.
- **Residual:** Delta-based change prediction.
- **Sequence (Flagship):** Multi-frame output generation.
